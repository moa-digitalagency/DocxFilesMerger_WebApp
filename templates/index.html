Medical Records Processor

This tool processes a ZIP archive containing medical records (.doc or .docx files), merges them into a single document, and converts it to PDF.

How it works:
  1. Upload a ZIP file containing .doc or .docx medical records
  2. The system will extract and merge all records into a single document
  3. Each record will be separated by a header line with the filename
  4. Download the merged document in DOCX and PDF formats

Instructions
Preparing Your ZIP File

The ZIP file should contain medical record documents in .doc or .docx format. The system will:

  • Extract all .doc and .docx files (ignoring other file types)
  • Merge the contents into a single document
  • Add a separator line before each document's content with the format:
    <FILENAME.extension>........................................................................................................
  • Generate both .docx and .pdf versions of the merged document
Important: For large archives with many files (5,000+), processing may take several minutes.
After Processing

Once processing is complete, you'll be able to download:

  • A merged .docx file containing all records
  • A .pdf version of the merged document
